===========================================
WIST CHROME EXTENSION - INSTALLATION GUIDE
===========================================

QUICK START:
------------
1. Open Chrome and go to: chrome://extensions/
2. Enable "Developer mode" (top right toggle)
3. Click "Load unpacked"
4. Select this folder (wist-extension-download)
5. Done! The extension is now installed.

HOW TO USE:
-----------
- Visit any product page (Amazon, Target, Etsy, etc.)
- Click the Wist extension icon in your toolbar
- Click "Save to Wishlist" to add the product

REQUIREMENTS:
------------
- Google Chrome browser
- Account on https://wishlist.nuvio.cloud

For detailed instructions, see INSTALLATION.md
